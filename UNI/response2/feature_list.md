

# 質問，依頼等の検出
## 前向き機能
- 発話内容に沿った応答を相手に働きかける機能
    - 質問
    - 依頼
    - 提案
    - 挨拶

## 後ろ向き機能
- 前向き機能を有する発話に対応する機能
    - 質問に対する応答
    - 対話内容に対する同意
    - 理解を伝える応答
    - 相互新年を形成する基盤化

------------

### 課題
- ラベリングの基準
    - 前向き機能 : forward
        | 項目              | ラベル | 
        | :---------------- | ------ | 
        | 質問 yes No       | 0      | 
        | 質問 What         | 1      | 
        | 依頼              | 2      | 
        | 提案              | 3      | 
        | 挨拶              | 4      |
        | 情報開示         | 5     | 
    
    - 後ろ向き機能 : backward

        | 項目              | ラベル | 
        | :---------------- | ------ | 
        | 質問の応答 yes No | 0      | 
        | 質問の応答 What   | 1      | 
        | 同意              | 2      | 
        | 理解              | 3      | 
        | 基盤化            | 4      | 
    

### メモ
- 対話行為

    | 対話行為    | 対応            | 
    | ----------- | --------------- | 
    | 質問(YesNo) | 前向き ：質問   | 
    | 質問(What)  | 前向き ：質問   | 
    | 応答(YesNo) | 後ろ向き : 応答 | 
    | 応答(平叙)  | 後ろ向き : 応答 | 
    | フィラー    | 対応なし        | 
    | あいづち    | 弱い基盤化      | 
    | 確認        | 基盤化，理解    | 
    | 要求        | 前向き ：依頼   | 

    - 挨拶，提案に対応するものがない
    - 同意もない


### 特徴量候補
- 質問 Yes No
    - 単語N-gram, 自立語, 文末n-gram，文末付属語列，文末n-gramペア，キーワード，自立語の有無，
- 質問 What
    - 単語N-gram，文末n-gram，文末付属語列，キーワード，文末n-gramペア，自立語の有無
- 要求
    - 文末が命令形，動詞基本形+"な"の命令形，動詞連用形+"て"+{"よ"|"ね"}，動詞連用形+"や"+{"よ"|"ね"}
- 依頼
